# blog.mtib.dev

This repo serves as a hub to all the places my stuff can be found, a blog for low-effort publishing and a playground. For proper polished front-end code look elsewhere ;)

```sh
yarn
yarn start
yarn build-dev
yarn build-prod
yarn deploy
```

## Troubleshooting

Running on M1 might require:

```sh
arch -x86_64 zsh
npm i -g node-gyp
```
